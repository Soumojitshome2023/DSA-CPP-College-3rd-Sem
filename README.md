## **Command :**

Create a Directory :

```
mkdir directory_name
```

Create File :

```
pluma fileName
```

Change Directory :

```
cd /path/to/directory
```

Compile C++ :

```
g++ fileName.cpp
```

Run :

```
./a.out
```

or

```
./a.exe
```
